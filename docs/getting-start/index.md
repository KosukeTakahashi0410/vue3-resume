# はじめに

## 環境構築

### Volta をインストールする

?> Node と yarn を使用使いますが、バージョン管理などを楽にするため Volta を推奨します。

[Volta installation](https://docs.volta.sh/guide/getting-started)

Volta をインストール後、yarn を実行してください。  
（Volta が正常にインストールされていれば問題なく行えるはずです。）

```sh
$ yarn
```
